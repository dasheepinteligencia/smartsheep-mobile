## Escopo observado

O app é um aplicativo Expo/React Native para operação em campo, com login, roteiro de visitas, execução de check-in/check-out/justificativas, pesquisas, evidências por foto, recursos offline com SQLite e sincronização com API.

Quando algum comportamento não pôde ser confirmado diretamente no código analisado, este documento usa a marcação **A confirmar**.
